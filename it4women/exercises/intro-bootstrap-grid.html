The Bootstrap Grid

Learning Goals

At the end of this Tutorial you will be able to:

  • Create and use responsive one, two, three and four-column grids with Bootstrap.

Adding the Bootstrap container class

Column layout in Bootstrap is based upon a CSS class named container that is set inside the Bootstrap stylesheet file. Let's add this CSS class to your web page.

  1. Enclose the placeholder text, of a heading and a paragraph, within a pair of opening and closing <div> ... </div> tags. See below.
  2. Give your opening <div> tag a class name of container, as shown below.
  3. Save your bootstrap-grid.html file. In your web browser, it should look as shown below.

Note how the layout of your web page content has changed by adding the Bootstrap CSS class of container.

Bootstrap has added some blank spacing around the four edges of your content. As a result, text, images or other content inside the container no longer push directly against the sides of the web browser window.

To help you visualise this extra spacing, it is highlighted in pink below.

Boostrap will adjust this extra spacing responsively according to the wide of the screen.

Without the need to add any further layout tags to a web page or extra classes to a stylesheet, the Bootstrap container provides a convenient way of creating single-column layouts on web pages.

While the container class takes care of content layout or 'positioning', you will, of course, still need to add further CSS classes in the your own stylesheet to set fonts and colours.

Using the VS Code visual indentation guides

Visual Studio Code displays vertical lines called indentation guides to help you visually organise blocks of content in your web pages.

Indented text is text that is ‘pushed in‘ from the left margin, typically by pressing the Tab key.

Indentation guides help you identify errors such as missing (or extra) opening or closing HTML tags. In the example below, you can see that the opening and closing <div> ... </div> tags line up correctly.

Always use indentation when working with Bootstrap web pages. Here are the steps.

  1. In your bootstrap-grid.html file, use the mouse to drag over and select the placeholder text inside the container tag pair. See below.
  2. Press the Tab key to indent the placeholder text. See below.
  3. Save your bootstrap-grid.html file.

Creating a two-column layout in Bootstrap

When creating a single-column layout in Bootstrap, it is enough to use the container class.

To create multi-column layouts of two, three or more columns, however, you need to use another Bootstrap CSS class called row.

The following exercise provides an example.

  1. Copy the following placeholder text. 
     <div class="container">
    <h2>Sub-heading</h2>
    <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nulla exercitationem ipsum enim natus soluta rem eos? In dicta eius, reprehenderit illo odit mollitia nulla ex et odio minus expedita numquam!</p>

</div>

      Making your Bootstrap columns responsive

      In Bootstrap, columns you create with the class of col are not responsive to the size of the user's display device.

      For example, the layout you created above will appear as two columns on both mobile and desktop/laptop screens.

      Typically, you will want your layouts of content to appear as:

      • Two, three or four columns on wide screens (768px or wider)
      • Single columns on narrow screens (less than 768px).

      The Bootstrap grid system is based on 12 columns. This means:

      • A two-column layout is really two sets of six Bootstrap columns (2 x 6 = 12).
      • A three-column layout is really three sets of four Bootstrap columns (3 x 4 = 12).
      • A four-column layout is really four sets of three Bootstrap columns (4 x 3 = 12).

      To make the above two-column layout responsive, replace the col class names with the col-md-6 names as shown below.

Creating a three-column layout in Bootstrap

Next, we will create a responsive three-column layout in Bootstrap. Here are the steps:

  1. In the bootstrap-grid.html web page, under the existing content, copy-and-paste the following: 
     <div class="container">
   <div class="row">
      <div class="col-md-4">
         <h2>Sub-heading</h2>
         <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nulla exercitationem ipsum enim natus soluta rem eos? In dicta eius, reprehenderit illo odit mollitia nulla ex et odio minus expedita numquam!</p>
