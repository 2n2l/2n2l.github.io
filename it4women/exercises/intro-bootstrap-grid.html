  1. In VS Code, open the bootstrap.html web page from your bootstrap sub-folder.
  2. Use the File | Save As command to save the file with the following new name:   bootstrap-grid.html
  3. Enclose the placeholder text, of a heading and a paragraph, within a pair of opening and closing <div> ... </div> tags. See below.
  4. Use the mouse to drag over and select the placeholder text. See below.
  5. Press the Tab key to indent the placeholder text. See below. Visual Studio Code displays vertical lines called indentation guides to help you visually organise blocks of indented text.   Indented text is text that is ‘pushed in‘ from the left margin, typically by pressing the Tab key. In the example below, you can see that the opening and closing <div> ... </div> tags line up correctly.
  6. Give your opening <div> tag a class name of container, as shown below.
  7. Save your bootstrap-grid.html file. In your web browser, it should look as shown below.

That’s it. You have successfully created a basic, empty web page.

Making your Bootstrap columns responsive

In Bootstrap, columns you create with the class of col are not responsive to the size of the user's display device.

For example, the layout you created above will appear as two columns on both mobile and desktop/laptop screens.

Typically, you will want your layouts of content to appear as:

  • Two, three or four columns on wide screens (768px or wider)
  • Single columns on narrow screens (less than 768px).

The Bootstrap grid system is based on 12 columns. This means:

  • A two-column layout is really two sets of six Bootstrap columns (2 x 6 = 12).
  • A three-column layout is really three sets of four Bootstrap columns (3 x 4 = 12).
  • A four-column layout is really four sets of three Bootstrap columns (4 x 3 = 12).

To make the above two-column layout responsive, replace the col class names with the col-md-6 names as shown below.

Creating a three-column layout in Bootstrap

Next, we will create a responsive three-columnn layout in Bootstrap. Here are the steps:

  1. In the bootstrap-grid.html web page, under the two-column layout, copy-and-paste the following:
